# 🥚 Crock N' Roll

Meal planning for people who don't want to think about it.

Auto-picks slow cooker meals, builds grocery lists, reminds you to shop. Multi-store support (Walmart, Target, Harris Teeter, Food Lion). AI-powered meal discovery. Zero decisions required.

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Netlify

Push this repo to GitHub, connect to Netlify. It builds automatically.

## Email Reminders

See `crockn_roll_email.py` for Gmail-based grocery reminders.
Requires a Gmail App Password — see instructions in the script.

## Built With

React + Vite (frontend) · Python (email cron)
